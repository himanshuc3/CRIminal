# CRIminal
Cricket scorecards at the feet of your terminal.Helps in getting rid of cricket addiction by sucking cricket out of it. As a consequence, saves money and time by removing unwanted subscriptions.

# How to run

1. Check and install python 3.x if not present:
 ```bash
  python -v # To check python version
 ```
Python binaries are present on https://www.python.org/downloads/
